# Multi-Click_Skill_Button--Ogylandy-Yespagambetov

## Thresholds

Long Press Threshold - 0.5f(can be togled)
Double Click Threshold - 0.3(can be togled)

## Progress through phases

If nothing is happening - idle

If the player presses and releases quickly one time - Short Click

If the player keeps holding the button for longer than the set time - a Long Press

If the player presses the button two times quickly - Double Click
